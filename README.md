This is a Brainknot interpreter written in Python.

You can use it as a module:
```python
  from brainknot import brainknot

  sourcecode = """
    *(>[>*,>]<[,*])
  """

  inputs = "011011"

  out = brainknot(sourcecode,inputs)
  print(out)
```
